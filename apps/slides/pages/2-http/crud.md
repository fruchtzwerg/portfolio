# CRUD

Create, Read, Update, Delete

## HTTP Methods

- **POST** - Create
- **GET** - Read
- **PATCH** - Update
- **DELETE** - Delete
- **PUT** - Replace/Upsert *"Update or Insert"*

<br>

<v-click>

- **OPTIONS** - Get available HTTP methods for a <span class="text-teal-400">resource</span>
- **HEAD** - Same as GET, but only returns the headers

</v-click>
