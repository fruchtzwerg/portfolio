# HTTP Verbs

**CRUD** - **C**reate, **R**ead, **U**pdate, **D**elete

- **POST** - Create
- **GET** - Read
- **PATCH** - Update
- **PUT** - Upsert/Replace
- **DELETE** - Delete

<br>

<v-click>

- **OPTIONS** - Get available HTTP methods for a <span class="text-teal-400">resource</span>
- **HEAD** - Same as GET, but only returns the headers

</v-click>
